# PrimeLink Website

Moderna web stranica za PrimeLink d.o.o. Zagreb.

## Tehnologije

- Vite
- TypeScript
- React
- Tailwind CSS
- shadcn/ui
- i18next
- Framer Motion

## Razvoj

```sh
# Instalacija ovisnosti
npm install

# Pokretanje razvojnog poslužitelja
npm run dev

# Build za produkciju
npm run build
```
